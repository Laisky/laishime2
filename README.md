### Deploy

#### Local

```sh
$ cd laishime2/1
$ python bootstrap.sh
$ bin/buildout install
$ bin/python setup.py develop
$ bin/python -m laishime
```

#### SAE

原来就是打算在 SAE 的，直接上传 laishime/1 到 SAE 就可直接运行
